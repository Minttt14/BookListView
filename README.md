# 圖書管理系統

## 主要功能

內建 12 本涵蓋章回小說、詩歌、戲曲與史書的經典文學名著，可自由切換五種檢視模式來瀏覽館藏，並透過「雙擊操作」完成借閱與取消借閱，系統具備完善的防呆機制，能防止重複借閱同一本書籍，並即時更新右下方的借書清單。

## 使用方法

1.程式啟動後，左側主畫面預設會以「大圖示」載入 12 本書籍，可以點擊右上角的「檢視方式」下拉式選單，自由切換為「大圖示」、「詳細資料」、「小圖示」、「清單」或「大圖示加詳細資料」等五種排版模式

2.點擊兩下想借閱的書籍會彈出確認視窗，詢問「確定要借閱嗎?」，視窗標題會顯示選取的書名，點選「是」之後，該本書籍就會成功借出，並自動新增至右下方的「借書清單」列表內

<img width="350" height="200" alt="image" src="https://github.com/user-attachments/assets/95e1fd12-057f-452c-a97c-c783506dc1d5" />
<br>

3.若想取消借閱書籍，點擊兩下右下角借書清單中的書名，會跳出取消借閱的通知視窗，點選「是」之後，該本書籍就會取消借閱，並從右下方的「借書清單」列表內消失

<img width="350" height="210" alt="image" src="https://github.com/user-attachments/assets/32c95a2d-3a31-4e21-b8dc-6a5338681fed" />
<br>

4.為避免操作失誤導致清單混亂，系統內部已建立自動查核機制，若雙擊了一本「已經存在於借書清單中」的書籍，系統會自動攔截動作，不會再次跳出借閱確認視窗，也不會將重複的書名加入清單中，確保借閱紀錄正確無誤
